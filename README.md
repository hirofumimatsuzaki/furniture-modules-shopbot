# ShopBot Modular Parts Generator

合板の廃材を前提に、モジュール家具用の部品を `p5.js` で生成して `SVG` 出力するツールです。

## 生成モード
- `モジュール` タブ:
  - `PANEL`: 4辺にスロット付きの正方形パネル
  - `BRACE`: 細長い補強材（上下スロット）
  - `CORNER`: L字コーナーパーツ
- `Box` タブ:
  - 6面構成の `BOXx-TOP/BOTTOM/FRONT/BACK/LEFT/RIGHT` をセット生成
  - 各辺を凹凸のフィンガージョイント（はめ込み）形状で作成
  - `上面に蓋パネルを付ける` をOFFにすると `TOP` を生成せず、側板上辺をまっすぐにして棚用途に使いやすくする

## 使い方
1. `/Users/matsuzakihirofumi/testcodex/furniture-modules/index.html` をブラウザで開く
2. 左上タブで `モジュール` か `Box` を選択
3. 左パネルで以下を調整
- シートサイズ
- モードごとの寸法（パネル or Box幅奥行高さ）
- 板厚 / クリアランス / エンドミル径
- 部品間隔
4. `再生成` でレイアウト更新
5. `SVG保存` で書き出し

## 加工メモ
- 単位は `mm`
- スロット幅は `板厚 + クリアランス`
- スロット内角逃がしとして簡易 dogbone 円を出力
- CAM 側で外形/内形の工具経路を割り当ててください

## 注意
- 部品がシートに入り切らない場合、画面上の `dropped` に未配置数が出ます
- 最終寸法は必ず端材実測と試し切りで補正してください
